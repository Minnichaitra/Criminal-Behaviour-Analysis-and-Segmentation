# Criminal Behaviour Analysis and Segmentation

## Overview
This project analyzes criminal behaviour patterns and segments districts based on crime statistics using Machine Learning clustering techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Criminal behaviour analysis
- District crime segmentation
- K-Means clustering
- Crime rate visualization
- Data preprocessing and analysis

## Algorithm Used
K-Means Clustering

## Project Workflow
1. Data collection
2. Data preprocessing
3. Feature scaling
4. K-Means clustering
5. Crime behaviour visualization
6. Cluster analysis
